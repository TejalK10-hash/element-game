# 🚀 Element Navigator: Connect the Tech

An interactive educational web game that teaches how real-world elements and advanced materials are used in aerospace engineering.

Players connect materials such as **Aluminum-Lithium Alloy, Niobium, Copper, Hafnium Carbide, Xenon, Gold, and Plutonium-238** to the spacecraft, aircraft, and probe components that rely on them. Each successful match reveals educational facts explaining why that material is critical to the technology.

---

## 🎮 Features

- Interactive drag-and-drop gameplay
- Four aerospace-themed levels:
  - 🚀 Rocket
  - 🛰️ Space Shuttle
  - ✈️ Fighter Jet
  - 🔭 Space Probe
- Educational material science content
- Dynamic scoring system
- Level progression
- Completion tracking
- Material fact popups
- Final glossary review
- Responsive modern UI
- Support for custom spacecraft and material imagery

---

## 📚 Learning Objectives

Players learn:

- Why specific materials are selected for aerospace applications
- Thermal protection systems
- Rocket propulsion materials
- Spacecraft structural materials
- Deep-space power systems
- Hypersonic flight technologies
- Material properties such as:
  - Heat resistance
  - Strength-to-weight ratio
  - Corrosion resistance
  - Thermal conductivity
  - Radiation durability

## 🖼️ Custom Images

The game supports custom images for each vehicle.

Add the following files to the project folder:

- `rocket.png`
- `shuttle.png`
- `jet.png`
- `probe.png`

If an image is not found, the game automatically displays a placeholder frame.

---

## 🎯 Gameplay

1. Select or drag a material badge.
2. Connect it to the correct component on the vehicle.
3. Receive educational feedback.
4. Complete all matches in a level.
5. Progress through all aerospace systems.
6. Unlock the final glossary and score summary.
